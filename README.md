# Gradify

A lightweight Go library to create colorful terminal gradiant text.

It converts hex colors to RGB and uses linear interpolation for vibrant multi-color gradients in your terminal.

---

## Features

- Hex to RGB conversion  
- Multi-color linear interpolation  
- Built-in color presets  
- Truecolor ANSI support  
- Simple API

---

## Installation

```bash
go get github.com/yourpov/gradify
```
